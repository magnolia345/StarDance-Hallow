Hollow
A first-person 3D survival horror game made in Godot 4. No combat. No weapons. Just you, a flickering flashlight, and whatever is in the walls.
About
  HOLLOW is a short-form horror game built for the Stardance Summer Program. Inspired by Resident Evil 7, Puppet Combo games (Stay Out of the House, Nun Massacre), The Evil Nun, and Granny. This game strips survival horror down to its core: explore, find the thing, get out alive. (lol sorry for the lack of formatting, I'll fix that soon)
